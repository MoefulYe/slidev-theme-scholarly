# 贡献指南

我们欢迎贡献！要开发这个主题：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 将演示导出为 PDF
npm run export

# 生成预览截图
npm run screenshot
```

编辑 `example.md` 或 `example-zh.md` 来测试你的更改.
