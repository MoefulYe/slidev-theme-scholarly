# 布局

本主题提供 20+ 种专业布局，专为学术演示设计。

## 布局分类

| 分类 | 用途 | 布局 |
|------|------|------|
| [结构布局](./structure) | 组织演示流程 | cover, section, agenda, end |
| [内容布局](./content) | 展示信息 | default, two-cols, figure, split-image |
| [强调布局](./emphasis) | 突出重点 | center, focus, quote, statement, fact |
| [学术布局](./academic) | 研究专用内容 | intro, methodology, results, references |

## 快速参考

### 结构布局

- **cover** - 标题页，含作者和机构信息
- **section** - 章节分隔页
- **agenda** - 演示大纲
- **end** - 结束页

### 内容布局

- **default** - 标准内容页（自动应用）
- **two-cols** - 双栏布局
- **figure** - 居中图片配说明
- **split-image** - 内容配侧边图片

### 强调布局

- **center** - 居中内容
- **focus** - 单一焦点信息
- **quote** - 引用展示
- **statement** - 关键陈述
- **fact** - 数据/事实突出

### 学术布局

- **intro** - 引言页
- **methodology** - 研究方法
- **results** - 研究发现
- **references** - 参考文献展示
- **acknowledgments** - 致谢

## 使用示例

```markdown
---
layout: cover
---

# 我的演示标题
副标题

---
layout: section
---

# 第一部分：引言

---
layout: two-cols
---

# 双栏内容

::left::
左侧内容

::right::
右侧内容
```

查看各布局分类页面了解详细文档。
