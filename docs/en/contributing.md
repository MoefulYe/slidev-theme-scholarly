# Contributing

We welcome contributions! To develop this theme:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Export presentation as PDF
npm run export

# Generate preview screenshots
npm run screenshot
```

Edit `example.md` to test your changes.
